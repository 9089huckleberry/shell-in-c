shell for unix - Developed a custom Unix-like shell in C to illustrate key concepts of process creation and command execution. This shell reads user input, parses it into commands and arguments, and supports built-in commands such as exit. It leverages system calls like fork() and execvp() for spawning child processes and executing external programs. The shell also features dynamic memory management for handling user input and argument parsing, with custom functions designed to tokenize input and manage memory allocation efficiently.

   shell for windows - Developed a custom Windows shell in C to demonstrate key concepts of command execution and argument parsing. This shell reads user input, parses it into commands and arguments, and supports basic built-in commands like exit. It utilizes the system() function to execute external commands and programs. The shell incorporates dynamic memory management for input handling and custom functions for tokenizing and interpreting user commands.
